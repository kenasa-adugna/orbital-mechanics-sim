# Orbital Mechanics Simulator

A Python simulation that uses numerical integration to model the trajectory of a satellite orbiting a planet.

## Features
- **Newtonian Gravity:** Uses the Universal Law of Gravitation.
- **Real-time Animation:** Visualized using Matplotlib.
- **Modular Design:** Separate physics engine for easy scaling.

## Installation
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/orbital-mechanics-sim.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the sim: `python main.py`
